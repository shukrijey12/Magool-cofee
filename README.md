# ☕ Magool Coffee

A modern and fully responsive Coffee Shop website built using **HTML5, CSS3, and JavaScript (ES6)**.

Magool Coffee is a front-end web project designed to demonstrate responsive web design and modern JavaScript concepts. The website provides users with an interactive experience, including menu filtering, drink searching, favorites, contact form validation, and dynamic coffee data loaded from a JSON file.

---

## 🚀 Features

* 🏠 Responsive Home Page
* ☕ Coffee Menu
* 🔍 Search Drinks
* 🗂️ Filter Drinks (All, Hot, Cold, Shakes)
* ❤️ Favorites using LocalStorage
* ☕ Coffee of the Day
* 📊 Animated Statistics Counter
* 📱 Mobile Responsive Navigation
* 📞 Contact Page
* ✅ Advanced Contact Form Validation
* 💾 LocalStorage Integration
* 📂 Coffee data loaded from **data.json**
* ⚡ Fetch API with Async/Await

## 📦 JSON Data

The project uses a local **json.js** file to store coffee information.

Example fields:

* id
* name
* price
* category
* image
* sold

The **Coffee of the Day** section automatically reads the coffee with the highest **sold** value using Fetch API and displays it on the Home page.
